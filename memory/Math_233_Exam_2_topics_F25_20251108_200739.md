## Math 233 Exam 2 Topics

## Fall 2025

Below is a list of important topics to know for Exam 2. Be sure to check out the practice exams and review your WebAssign homework.

- 14.6 Directional Derivatives and Gradient Vector
1. The directional derivative
2. The gradient vector
3. The direction and magnitude of greatest increase
4. Finding the tangent plane or normal line using the gradient vector
- 14.7 Maximum and Minimum Values
1. Local maximum and minimum values
2. Critical points, saddle points
3. Second derivatives test
4. Absolute maximum and minimum values
- 14.8 Lagrange Multipliers
1. Find minimum and maximum values of two or three variable functions using Lagrange multipliers with one constraint only
2. Not covered: Lagrange problems with two constraints
- 15.1 Double Integrals over Rectangles
1. Estimating volumes and values of double integrals using Riemann sums
2. Iterated integrals over rectangles.
3. Evaluating a double integral of a function f(x,y) that can be factored as the product of a function of x and a function of y.

- 15.2 Double Integrals over General Regions
1. Sketch type I and type II regions and express them using appropriate inequalities.
2. Set up double integrals over a general region, reverse the order of integration, and evaluate such integrals.
3. Finding area with double integrals
- 15.3 Double integrals in polar coordinates
1. Set up and sketch polar regions
2. Change to polar coordinates in double integrals; specifically, given a double integral in rectangular coordinates, recognize that it may be evaluated by changing to polar.
3. Finding areas enclosed by polar curves.
4. Finding volumes using double integrals in polar coordinates.
- 15.5 Surface Area
1. Finding the surface area of different surfaces over a given domain.
- 15.6 Triple integrals
1. Triple integrals over rectangular boxes and general regions
2. Changing the order of integration in a triple integral
3. Not covered: applications besides finding volume for triple integrals.
